Чем отличаются Web server и Application server?< /h1> <Р> Сервер приложений (англ. application server) — сервер, исполняющий некоторые прикладные программы. Сервер-приложений – объект, который обрабатывает запросы, связанные с приложениями, точнее для выполнения прикладных процессов (выборка данных, поиск данных, работа с терминалами). По идее эта технология изначально вообще не была связана с Web’om, однако, сейчас чаще говорят сервер web приложений. Практически используется для работы с базами данных. Веб-сервер — это сервер, принимающий HTTP-запросы от клиентов, обычно веб-браузеров, и выдающий им HTTP-ответы.Web-сервер – объект, который обрабатывает запросы, в частности http-запросы. Браузер в данном случае клиент, который делает запросы (POST, GET). < / p> < / body> < / html> © 2021 GitHub, Inc.
